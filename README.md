# Cloud Security Fundamentals — Documentation Project

A structured reference documenting core cloud security concepts across **AWS**, **GCP**, and **Azure**, built as part of my Cloud Technology & Information Security specialization at ITM Vocational University.

This project maps foundational security models (Shared Responsibility Model, CIA Triad) to real-world cloud services, and includes a practical security audit checklist that a Cloud Security Analyst or SOC Analyst would use in an entry-level role.

## Why this project

Cloud providers document their services individually, but rarely map them side-by-side against the security *principles* they support. This repo bridges that gap — connecting theory (what we study) to practice (what's configured in the console).
